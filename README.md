# Proyecto-AA-EMG_sensors

## Descripción
Este es el repositorio del trabajo final para la asignatura de Aprendizaje Avanzado. En este trabajo abordamos el problema de la interpretación de señales sEMG con el objetivo de clasificarlas en gestos dependiendo de sus características mediante el uso de diversos modelos. También realizamos análisis exploratorios de los datos para entenderlos y explicarlos mejor así como un estudio de las técnicas usadas en trabajos relacionados.

## Instalación

```bash
pip install -r requirements.txt
python download_dataset.py
```

## Ejecución

```bash
# Open the notebooks to run them

python lstm.py # For raw data
python lstm.py # For FD feature data
```
